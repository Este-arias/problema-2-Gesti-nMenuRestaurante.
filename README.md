## Descripción
Este proyecto fue desarrollado en Python como solución al Problema 2 de la Fase 5 - Evaluación Final POA.

El programa permite:

- Registrar 6 productos de un menú.
- Almacenar la información en una matriz.
- Aplicar un descuento del 15 % a los productos que pertenezcan a una categoría específica y cuyo precio base sea superior al umbral establecido.
- Mostrar el precio base, si aplica descuento y el precio final de cada producto.
